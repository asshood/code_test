Программа позволяет получить данные о твиттах по запросу в поисковой строке в категории Top(популярные) или  Latest(последние)
Все данные сохраняються в формате json
Программа будет собирать данные пока вы ее не выключите
Так же необходим иметь подключение через VPN для получение доступа к twitter, скачайте vpn версию для компьютера
Программа предназначена только для английской версии firefox!!!

Порядок Установки:
1. Установите библиотеку selenium
2. скачайте geckodriver для вашей версии firefox https://github.com/mozilla/geckodriver/releases
3. geckodriver переместите в папку с вашим проектом python
